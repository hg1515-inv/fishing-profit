# 🎣 Fishing Profit（釣果管理アプリ）プロジェクト概要

## 1. アプリケーションの目的
 Fishing Profitは、漁師や遊漁船運営者が日々の釣行における利益を簡単に記録・管理するための「収支管理ツール」です。
 売上から燃料代を差し引いた「純利益」を可視化し、安定した運営をサポートすることを目的としています。

---

## 2. 主要機能
- **日付ナビゲーション**:
  - 「◀」「▶」ボタンで前後の日に素早く移動。
  - カレンダー形式の日付選択機能（Date Picker）で任意の日に一瞬でジャンプ。
- **今日の釣行入力**:
  - 表示中の日付に連動してデータを保存・修正。
  - 入力中にリアルタイムで予想収益を表示するプレビュー機能。
- **今日の収支確認**:
  - 当日の利益、総売上、燃料代をひと目で確認可能。
- **月次統計・履歴**:
  - 今月全体の純利益、総売上、合計釣行回数を自動集計。
  - 過去10件の最新履歴をリスト表示。
- **データ永続化**:
  - ブラウザの `localStorage` を利用し、オフライン環境でもデータの保存・閲覧が可能。

---

## 3. デザインコンセプト（Premium Aesthetics）
- **モバイルファースト**: スマートフォンでの操作を前提としたシングルカラムレイアウト。
- **モダンなUIレイアウト**: 丸みを帯びたカード、クリーンなドロップシャドウ、洗練されたタイポグラフィ（システムフォント活用）を採用。
- **直感的な色彩**:
  - **Blue系**: 信頼感と「海」をイメージ。
  - **Success Green / Danger Red**: 利益と損失を視覚的に強調。

---

## 4. 技術スタック
- **Frontend**: React (Create React Appベース)
- **State Management**: React Hooks (useState, useEffect)
- **Styling**: Vanilla CSS-in-JS (Inline Styles)
- **Storage**: ブラウザ標準ローカルストレージ

---

## 5. 実行方法
プロジェクトディレクトリにて以下のコマンドを実行します。
```bash
npm run dev
```
※標準の `npm start` のエイリアスとして設定されています。
