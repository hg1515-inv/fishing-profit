# 🎣 Fishing Profit（釣果管理アプリ）

釣行ごとの売上・燃料代・純利益を管理するためのモバイルレディなウェブアプリケーションです。

## 🚀 はじめに

このアプリをローカルで起動するには以下のコマンドを実行してください：

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) が自動的に開きます。
